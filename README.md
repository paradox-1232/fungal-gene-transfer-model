# Fungal Gene Transfer Model

This project simulates horizontal gene transfer in a simple fungal-like network using Python.

## Description
A grid-based model is used where each cell represents a fungal unit. Cells can transfer genetic material to neighboring cells with a certain probability.

## Objective
To understand how genes spread across a network over time.

## Tools Used
- Python
- NumPy
- Matplotlib

## Output
The model generates a plot showing how gene presence increases over time.
